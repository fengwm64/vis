# PRD: Kosaraju 强连通分量

## 算法定义

Kosaraju 算法用于求有向图的强连通分量（SCC）。核心思路：

1. 对原图 G 做一次完整的 DFS，按完成时间压栈（后完成的在栈顶）。
2. 构造反图 G^T（所有边方向反转）。
3. 按栈顺序依次在反图上做 DFS，每次 DFS 访问到的节点构成一个 SCC。

## 输入规模与边界

- 节点数：默认 6-8 个，用户可通过预设切换不同图结构
- 边数：有向边，允许自环和重边
- 节点标签：单字母 A、B、C、...
- 输入数据由算法模块内置多组预设图，前端可切换

## 示例数据

预设 1（经典示例）：
```
节点: A, B, C, D, E, F
边: A→B, B→C, C→A, B→D, D→E, E→F, F→D
SCC: {A,B,C}, {D,E,F}
```

预设 2（单节点 SCC）：
```
节点: A, B, C, D
边: A→B, B→C, C→D
SCC: {A}, {B}, {C}, {D}
```

预设 3（全连通）：
```
节点: A, B, C
边: A→B, B→C, C→A
SCC: {A,B,C}
```

## 可视化步骤

动画分三个阶段，每阶段可独立单步/自动播放：

### 阶段 1：第一次 DFS（原图）
- 从某个起始节点开始 DFS
- 高亮当前访问的节点和遍历边
- 节点显示发现时间 / 完成时间
- 完成的节点按完成顺序压入栈（可视化栈）
- 重复直到所有节点完成

### 阶段 2：构建反图
- 动画展示所有边方向反转
- 原图淡出，反图淡入

### 阶段 3：第二次 DFS（反图）
- 按栈顶顺序依次弹出节点
- 从未访问的栈顶节点开始 DFS
- 每次 DFS 访问的节点用相同颜色标记为一个 SCC
- 所有节点标记完毕后展示最终 SCC 结果

## 复杂度说明

- 时间复杂度：O(V + E)，两次 DFS
- 空间复杂度：O(V)，栈和访问标记
