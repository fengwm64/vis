## PRD 备注：MST 权重

PRD 声称 MST 总权重为 16，实际正确值为 17（B-C=1, A-C=2, D-E=3, B-D=5, D-F=6）。PRD 中的 MST 边列表含 A-B(4)，但 A-B 在算法中被拒绝（A 和 B 已通过 A-C-B 连通）。算法实现正确，PRD 文档有误。
